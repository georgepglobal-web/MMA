# Fighter Avatar Images

## Required Images

Place 4 fighter avatar images in this directory with the following naming convention:

1. `fighter-novice.png` - Novice level avatar
   - Lean but underdeveloped physique
   - Narrow shoulders, limited muscle definition
   - Neutral posture
   - Plain rashguard

2. `fighter-intermediate.png` - Intermediate level avatar
   - Noticeable muscle growth (shoulders, chest, arms)
   - Improved posture
   - Subtle vascularity / definition
   - Better-fitted gear

3. `fighter-seasoned.png` - Seasoned level avatar
   - Clearly athletic build
   - Visible muscle separation (delts, chest, arms)
   - Slightly wider frame
   - Signs of training (tape, light scars, worn gloves)

4. `fighter-elite.png` - Elite level avatar
   - Peak athletic physique
   - Broad shoulders, thick neck, powerful arms
   - Lean and conditioned (not bulky)
   - Calm, dominant presence
   - Premium-looking gear

## Image Specifications

- **Format**: PNG with transparent background (recommended) or solid dark/neutral background
- **Aspect Ratio**: Square (1:1)
- **Resolution**: Minimum 512x512px, recommended 1024x1024px or higher
- **Style**: Semi-realistic illustration or high-quality stylized render
- **Content**: Front-facing bust (head + shoulders), same character, same pose, same camera angle
- **Lighting**: Consistent across all images
- **Character**: Neutral ethnicity, consistent facial features

## Image Requirements

All avatars must:
- Be the same character, same pose, same camera angle
- Be front-facing bust (head + shoulders)
- Maintain consistent lighting and framing
- Avoid exaggerated "bodybuilder" proportions
- Look like real fighters, not superheroes
- Focus on physique and posture progression

## Fallback Behavior

If images are not found, the application will display a gradient placeholder with the level initials.
